## tutorial
1. download it [here](https://github.com/KhanhNguyen9872/deobf_pyobfuscate.com/archive/refs/heads/main.zip) or git clone it!
2. unzip it!
3. run 'pyd_pyobf_deobf.py' file
4. replace [obfuscate] var from (obf file) to (tool deobf)
5. done!
